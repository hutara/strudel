مرحله ۱: شروع با Strudel REPL

Strudel چیست؟
Strudel یک محیط لایوکدینگ موسیقی مبتنی بر وب است که از الگوی TidalCycles الهام گرفته و با جاوااسکریپت و Web Audio کار می‌کند. بدون نیاز به نصب، مستقیماً در مرورگر اجرا می‌شود.

دسترسی آنلاین

Strudel WebApp

شروع سریع

اولین پترن 
// پخش نت C4 هر نیم‌ثانیه
note("c4").sound("piano")
نکته: از Ctrl+Enter برای اجرای کد و Ctrl+. برای توقف استفاده کنید.

مرحله ۲: آشنایی با رابط کاربری

محیط Strudel

ویرایشگر کد | دکمه Play | اسلایدر BPM | لیست نمونه‌ها

کلیدهای میانبر

کلید عملکرد توضیحات
Ctrl+Enter اجرای کد شروع پخش پترن
Ctrl+. توقف متوقف کردن تمام صداها
Ctrl+S ذخیره ذخیره به عنوان gist
Ctrl+L بارگذاری بارگذاری از gist
Ctrl+/ کامنت کامنت کردن خط

مرحله ۳: پترن‌های اصلی

پترن‌های ساده

sequence 
// پخش متوالی نت‌ها
note("c4 d4 e4 f4")
fast / slow 
// سرعت ۴ برابر
note("c4").fast(4)

// سرعت نصف
note("c4").slow(2)

پترن‌های تصادفی

rand / choose 
// عدد تصادفی بین 0-1
rand.range(0,1)

// انتخاب تصادفی از لیست
choose(["c4","eb4","g4"])

مرحله ۴: ترکیب با Stack

نکته: از stack() برای پخش همزمان چند پترن استفاده کنید.
ملودی + بیس + درام 
stack(
  note("c4 eb4 g4 bb4").sound("piano"),
  note("c2").sound("sawtooth").lpf(300),
  s("bd hh sd hh").fast(2)
)

مرحله ۵: Euclidean Rhythms

euclid(3,8) 
// 3 ضربه در 8 قدم
s("bd").euclid(3,8)

// با چرخش
note("c3").euclidRot(3,16,14)

جدول Euclidean

پترن hits steps نتیجه
euclid(3,8) 3 8 *--*--*-
euclid(5,8) 5 8 *-*--*-*
euclid(4,7) 4 7 *--*--*-

مرحله ۶: سینتی‌سایزرها و صداها

سینتی‌سایزرهای داخلی

sawtooth , Sine , square, triangle 
note("c4 eb4 g4").sound("sawtooth")
  .lpf(800).gain(0.7)

note("c3").sound("square")
  .delay(0.3).delaytime(0.25)

نمونه‌های صوتی

drum samples 
s("bd hh sd hh").bank("RolandTR909")
s("korg:3").fast(2)

مرحله ۷: افکت‌ها و پردازش

افکت پارامترها مثال
.delay() gain, time .delay(0.5).delaytime(0.25)
.reverb() room, damp .reverb(3)
.lpf() freq, q .lpf(800)
.hpf() freq .hpf(2000)

مرحله ۸: مثال‌های پیشرفته

مثال ۱: تکنو مینیمال

minimal_techno.strudel 
setBPM(130)

stack(
  s("bd").euclid(3,8),
  s("hh*8").gain(.3),
  note("/4").sound("sawtooth").lpf(400),
  note("c4 eb4 g4 bb4".slow(4)).sound("triangle").delay(.4)
)

مثال ۲: IDM پیچیده

idm_glitch.strudel 
setBPM(170)

stack(
  s("bd:2").euclid(5,16),
  s("~ [hh? hh]").fast(4).gain(rand.range(.2,.8)),
  note("/2").sound("fm").cutoff(sine.range(400,4000)),
  s("glitch:0").room(2).fast(rand.range(1,8))
)

مثال ۳: Ambient Drone

ambient_drone.strudel 
setBPM(60)

stack(
  note("/8").sound("sawtooth").lpf(200).gain(.8),
  note("".slow(16)).sound("sine").reverb(5).gain(.3),
  s("~ noise").hpf(8000).gain(.1).delay(.7)
).room(3)
