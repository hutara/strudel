مرحله ۱: شروع با Strudel REPL

Strudel چیست؟
Strudel یک محیط لایوکدینگ موسیقی مبتنی بر وب است که از الگوی TidalCycles الهام گرفته و با جاوااسکریپت و Web Audio کار می‌کند. بدون نیاز به نصب، مستقیماً در مرورگر اجرا می‌شود.

دسترسی آنلاین

Strudel WebApp

شروع سریع

اولین پترن 
// پخش نت C4 هر نیم‌ثانیه
note("c4").sound("piano")
نکته: از Ctrl+Enter برای اجرا و Ctrl+. برای توقف استفاده کنید.

مرحله ۲: رابط کاربری

محیط Strudel

ویرایشگر کد | دکمه Play | اسلایدر BPM | لیست نمونه‌ها

کلیدهای میانبر

کلید عملکرد توضیحات
Ctrl+Enter اجرا شروع پخش
Ctrl+. توقف متوقف کردن صدا
Ctrl+S ذخیره gist
Ctrl+L بارگذاری gist
Ctrl+/ کامنت خط

مرحله ۳: پترن‌های اصلی

sequence

sequence 
note("c4 d4 e4 f4")

fast / slow

سرعت 
note("c4").fast(4)
note("c4").slow(2)

rand / choose

تصادفی 
rand.range(0,1)
choose(["c4","eb4","g4"])

مرحله ۴: ترکیب با Stack

ملودی + بیس + درام 
stack(
  note("c4 eb4 g4 bb4").sound("piano"),
  note("c2").sound("sawtooth").lpf(300),
  s("bd hh sd hh").fast(2)
)

مرحله ۵: ریتم‌های اقلیدسی

euclid

note("c3").euclidRot(3,16,14)

مرحله ۶: سینتی‌سایزرها

سینتی‌سایزرها 
note("c4 eb4 g4").sound("sawtooth").lpf(800)
note("c3").sound("square").delay(0.3)
نمونه‌ها 
s("bd hh sd hh").bank("RolandTR909")
s("korg:3").fast(2)

مرحله ۷: افکت‌ها

افکت پارامتر مثال
.delay() gain, time .delay(0.5).delaytime(0.25)
.reverb() room .reverb(3)
.lpf() freq .lpf(800)

مرحله ۸: مثال‌های پیشرفته

تکنو مینیمال

minimal_techno.strudel 
setBPM(130)

stack(
  s("bd").euclid(3,8),
  s("hh*8").gain(.3),
  note("/4").sound("sawtooth").lpf(400),
  note("c4 eb4 g4 bb4".slow(4)).sound("triangle").delay(.4)
)

IDM پیچیده

idm_glitch.strudel 
setBPM(170)

stack(
  s("bd:2").euclid(5,16),
  s("~ [hh? hh]").fast(4).gain(rand.range(.2,.8)),
  note("/2").sound("fm").cutoff(sine.range(400,4000)),
  s("glitch:0").room(2).fast(rand.range(1,8))
)
